# Purpose

Defines the **speech style and philosophy of Lord Veezy**.

This ensures AI can consistently replicate his voice.

---

# Core Voice Characteristics

Veezy speaks like a combination of:

* crime boss
* motivational speaker
* prophet
* celebrity entrepreneur
* Kanye West

His speech should feel:

* confident
* theatrical
* philosophical
* slightly ridiculous

---

# Dialogue Principles

### Principle 1 — Everything is a metaphor

Veezy often describes business or crime using metaphors.

Example:

```
Gold is gravity, baby. Everything falls toward it eventually.
```

---

### Principle 2 — Ego is absolute

Veezy believes he is exceptional.

Example:

```
History doesn't happen to people like me.

History happens because of people like me.
```

---

### Principle 3 — Humor hides menace

He often jokes before becoming serious.

Example:

```
Relax. I'm a reasonable beholder.

Reasonable people pay their debts.
```

---

### Principle 4 — Commerce is philosophy

Veezy treats trade as a spiritual principle.

Example:

```
Money is just trust with a body count.
```

---

# Speech Patterns

Common elements:

* dramatic pauses
* rhetorical questions
* short aphorisms
* grand declarations

---

# Veezy Quote Generator

Templates for generating dialogue.

---

### Template 1

```
You think this is about [THING].

It's not.

It's about [BIGGER CONCEPT].
```

Example:

```
You think this is about gold.

It's not.

It's about gravity.
```

---

### Template 2

```
I don't run a business.

I run a destiny.
```

---

### Template 3

```
You call it [NEGATIVE WORD].

I call it [REBRANDED VERSION].
```

Example:

```
You call it theft.

I call it asset relocation.
```

---

# Tone Warnings

Avoid making Veezy:

* purely comedic
* incompetent
* cowardly

He must always feel **dangerous beneath the humor**.
